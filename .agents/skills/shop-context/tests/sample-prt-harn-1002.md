# Sample PRT HARN-1002

Synthetic PRT / work instruction for `HARN-1002 Rev B`.

## Header

```text
Document: PRT-HARN-1002-B
Part: HARN-1002 Rev B
Revision Note: Rev B combines termination and sleeve operation.
```

## General Notes

- Keep traveler with the kit.
- Use current crimp pull-test chart.
- Heat tools must be returned to standby after use.
- Operator initials required after each operation.

## Op 0100 - Material Issue

- Issue wire, contacts, splice sleeves, heat shrink, and connector shells.
- Verify kit count.

## Op 0200 - Cut / Strip

- Cut all wires per cut list.
- Strip both ends per terminal table.
- Verify first-piece strip length.

## Op 0300 - Terminate / Sleeve Leads

- Crimp contacts to listed leads.
- Solder splice where required by splice table.
- Install sleeve over splice or termination area.
- Apply heat shrink using controlled heat gun.
- Verify insulation coverage and no exposed conductor outside allowed window.
- Pull-test sample crimps.

## Op 0400 - Harness Assembly

- Lay harness on build board.
- Route branches and install connector shells.
- Tie harness per layout notes.

## Op 0500 - E-Test

- Run continuity/electrical test.
- Verify no opens or shorts.

## Op 0600 - Final Insp

- Inspect labels, sleeve coverage, connector seating, branch lengths, and workmanship.
- Record acceptance on traveler.

## Op 0700 - Closeout

- Confirm all operation signoffs.
- Bag harness and close work order.
