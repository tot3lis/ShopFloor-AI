# Sample PRT HARN-1003

Synthetic PRT / work instruction for `HARN-1003 Rev C`.

## Header

```text
Document: PRT-HARN-1003-C
Part: HARN-1003 Rev C
Revision Note: Rev C splits cut and strip for long lead control.
```

## Op 0100 - Kit

- Pull wire, terminals, solder sleeves, labels, and connector kit.
- Verify wire gauge and color.

## Op 0200 - Cut Wires

- Cut wires to cut sheet.
- Bundle cut wires by circuit group.
- Verify length before moving to strip.

## Op 0250 - Strip Leads

- Strip insulation to listed length.
- Inspect for nicked strands.
- Recut and re-strip any damaged lead.

## Op 0300 - Install Terminals

- Crimp/install terminals on stripped leads.
- Verify terminal orientation.
- Pull-test sample crimps.

## Op 0400 - Solder Splices

- Solder splices where shown on splice chart.
- Verify wetting and no cold solder joints.

## Op 0500 - Sleeve

- Install sleeve over solder splice.
- Shrink sleeve until adhesive flow is visible where applicable.

## Op 0600 - Buzz Out

- Buzz out all circuits for continuity.
- Verify no shorts between adjacent circuits.

## Op 0700 - Final QA

- Inspect workmanship, sleeve position, connector orientation, labels, and branch dimensions.
- Sign final QA traveler block.
