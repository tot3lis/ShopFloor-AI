# Sample PRT HARN-1004

Synthetic PRT / work instruction for `HARN-1004 Rev A`.

## Header

```text
Document: PRT-HARN-1004-A
Part: HARN-1004 Rev A
Revision Note: New compact harness flow.
```

## Op 0100 - Kit Pull

- Pull wire, terminals, solder, sleeve, connector bodies, labels, and tie material.
- Verify kit against BOM.

## Op 0200 - Prep Leads

- Cut leads to released lengths.
- Strip insulation per termination table.
- Verify lead ID labels are correct.

## Op 0300 - Crimp / Solder

- Crimp terminals where specified.
- Solder splice joints where specified.
- Pull-test sample crimps.
- Inspect solder joints before insulation.

## Op 0400 - Insulate Splices

- Apply sleeve or heat shrink over splice areas.
- Heat shrink until fully seated.
- Verify insulation covers splice and does not interfere with connector assembly.

## Op 0500 - Board Layout

- Place harness on board.
- Route branches and lace/tie per board notes.
- Check connector clocking before removal.

## Op 0600 - Electrical Test

- Run electrical continuity test.
- Verify circuit ID against test printout when provided.

## Op 0700 - Final Inspection

- Final inspect labels, tie locations, connector orientation, sleeve coverage, and workmanship.
- Mark traveler accepted or rejected.
