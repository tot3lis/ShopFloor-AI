# Add ESD SME Scenario

## Input Scenario

Use this `shop-reference.md` and add an ESD SME.

## Expected Behavior

- ESD SME = User-requested
- Source depth = Level 1 unless supporting documents are provided
- Scope = shop-wide unless a specific operation is identified
- Suggested uploads are listed
- No claim that ESD is a standard operation

## Suggested Uploads

- ESD control plan
- ESD audit checklist
- wrist strap or footwear test procedure
- workstation requirements
- customer ESD requirements
