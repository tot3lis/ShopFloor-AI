## Output 2: `sme-registry.md`

```md
# SME Registry

| SME Name | SME Type | Status | Confidence | Source Depth | Source Basis | Related Operations | Related Machines / Capabilities | Exact / Named Triggers | Semantic / Plain-Language Triggers | Profile Path | Can Contribute Now | Needs Level-Up Documents |
|---|---|---|---|---|---|---|---|---|---|---|---|---|
```

Trigger columns are hints and summaries, not hardcoded gates.

Allowed SME Type values:

- Shop flow
- Inspection coordinator
- Machine / Capability
- User-requested

Allowed status values:

- Required
- Conditional
- Optional / Capability Available
- Needs Review
- User-requested
- Not Detected
