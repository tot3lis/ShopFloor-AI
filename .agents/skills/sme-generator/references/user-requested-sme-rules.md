# User-Requested SME Rules

Allow user-requested SMEs even when not generated from `shop-reference.md`.

When adding a user-requested SME:

- Set status to User-requested.
- Set SME Source Type to User-requested.
- Default source depth to Level 1 unless supporting source documents are provided.
- If `shop-reference.md` supports a machine/capability link, record that as a possible or conditional link.
- Do not claim the SME is part of standard flow unless the source directly supports it.
- Do not add deep technical knowledge, public references, RCCA logic, root cause analysis, corrective action, process advice, web scraping, or vector DB behavior.
