# Shop Reference Reading Rules

Read `shop-reference.md` as the source of shop-aware context. Preserve source wording exactly when copying operation numbers, operation names, work centers, machine names, evidence sources, and open questions.

## Sections To Extract

Extract these sections when present:

- Shop Overview
- Source Documents Reviewed
- Product Families
- Standard Operation Flow
- Operation Dictionary
- Work Center Dictionary
- Machine-To-Operation Map
- Manual / Human Operations
- Inspection And Test Points
- Likely Evidence Sources By Operation
- Shop-Specific Terminology
- Open Mapping Questions
- Revision Notes

If a section is missing, note it only when it affects confidence or creates an open question.

## Reading Priority

Use the strongest available source in this order:

1. Explicit Standard Operation Flow entry
2. Operation Dictionary entry
3. Machine-To-Operation Map link
4. Inspection And Test Points
5. Manual / Human Operations
6. Likely Evidence Sources By Operation
7. Work Center Dictionary
8. Shop-Specific Terminology
9. Open Mapping Questions
10. Generic inference from names

Operation evidence is stronger than machine-only evidence. Machine-only evidence can create Optional SMEs, but it cannot make an SME Required unless the source ties the machine to a standard operation.

## Preservation Rules

- Preserve operation numbers exactly.
- Preserve operation names exactly.
- Preserve work centers exactly.
- Preserve machine names exactly.
- Preserve unresolved wording exactly and add a separate note instead of rewriting it.

## Ambiguity Rules

Generic names such as `Process`, `Inspect`, `Test`, `Assembly`, or numeric-only work centers should not be forced into a specific SME. Assign Needs review or lower confidence and create an open question.
